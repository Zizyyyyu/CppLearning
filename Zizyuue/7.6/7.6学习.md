$$

$$

 
$$
\theta下的maxE[log π_θ(a_t | o_t)]是模型训练的时候取的最佳动作，因此损失函数通常设计为该值的相反数
$$
另一种方式是定义一个损失函数，
$$
c(s_t, a_t) =
\begin{cases}
0 & \text{if } a_t = \pi^\star(s_t) \\
1 & \text{otherwise}
\end{cases}
$$
进而转换为取该损失函数期望的最小值。**模仿学习中，如果数据集存在“出现错误”然后更正的数据，将会使模型的鲁棒性更强。**

通常，一段视频将被拆解为帧序列，通过**LSTM或者Transformer**进行训练

连续的分布的处理方法：**考虑Diffusion model(已严肃学习)**
$$
\begin{align*}
a_{t,0} &= \text{true action} \\
a_{t,i+1} &= a_{t,i} + \text{noise}
\end{align*}
$$

$$
\begin{align*}
\text{Learned network: } & f(s_t, a_{t,i}) = a_{t,i-1} \\
& \bigl(\text{actually use } f(s_t, a_{t,i}) = \text{noise}\bigr) \\
& a_{t,i-1} = a_{t,i} - f(s_t, a_{t,i})
\end{align*}
$$

提升imitation study的方法：**Multi-task training, Goal-conditioned behavioral cloning**

```python
import torch
def generate_batch(batch_size=256):
    """
    目的：生成一系列的模仿学习数据集
    :return:state,expert_action
    """
    state=torch.rand(batch_size, 2)*2-1
    goal=torch.zeros_like(state)
    expert_actions=goal-state
    return state,expert_actions
import torch.nn as nn
import torch.nn.functional as F
class BCNet(nn.Module):
    def __init__(self):
        super(BCNet, self).__init__()
        self.net=nn.Sequential(
            nn.Linear(2,1024),
            nn.ReLU(),
            nn.Linear(1024,64),
            nn.ReLU(),
            nn.Linear(64,2),
        )
    def forward(self,x):
        return self.net(x)
model=BCNet()
optimizer=torch.optim.Adam(model.parameters())
epoch=500
model.train()
for i in range(epoch):
    state,expert_actions=generate_batch()
    pred_action=model(state)
    loss=F.mse_loss(pred_action,expert_actions)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    print(loss)
state=torch.tensor([-1.2,1.3])
traj=[]
for t in range(300):
    with torch.no_grad():
        action=model(state)
        state=state+0.1*action
        traj.append(state)
print(f'final state:{state}')
print(traj)
"""final state:tensor([-0.0019,  0.0010])"""
```
