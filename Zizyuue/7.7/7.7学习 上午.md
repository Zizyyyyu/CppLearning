# 7.7 Goal-conditioned Behavior Cloning 实验笔记

## 1. 实验目标

本实验的目标是实现一个最小版的 **Goal-conditioned Behavior Cloning**。

让模型根据：

$$
state = [x, y]
$$

和目标点：

$$
goal = [g_x, g_y]
$$

输出动作：

$$
action = [d_x, d_y]
$$

也就是学习：

$$
state + goal \rightarrow action
$$

模型需要学会：

$$
[x, y, g_x, g_y] \rightarrow [g_x - x, g_y - y]
$$

---

## 2. 和普通 Behavior Cloning 的区别

普通 Behavior Cloning 是：

$$
state \rightarrow action
$$

例如固定目标为原点时：

$$
expert\_action = [0, 0] - state
$$

但 Goal-conditioned Behavior Cloning 是：

$$
state + goal \rightarrow action
$$

也就是目标点不再固定，而是作为模型输入的一部分。

因此，模型需要同时知道：

1. 当前在哪里；
2. 想要去哪里；
3. 应该怎么动。

---

## 3. 数据定义

在本实验中：

```python
state = [x, y]
goal = [gx, gy]
obs = [x, y, gx, gy]
expert_action = goal - state
```

```
观测 → 动作 → 状态更新 → 再观测 → 再动作
```
